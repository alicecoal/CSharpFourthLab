В XML и JSON 3 поля
SourceDirectory - исходная директория 
TargetDirectory - целевая дирректория(где хранятся архивы)
ArchiveDirectory - директория где хранятся по структуре год/месяц/день(дата закидывания файла определяется в программе и потом добавляется к строке)
К второй лабе добавлены 3 класса
Manager, SendingOptions, Converter

Converter парсит xml и json
И возвращает SendingOptions объект
Manager соответственно управляет этим
Если файлы не нашлись, то по умолчанию закидываются стандартные настройки
